a managed bean can be injected at client code' constructor by placing @Inject annotation
    
        @DojoComponent
        public class MyGreeter{
            public String sayHi(){
                return "hello from MyGreeter";
            }
        }

        public class Baz {
            private final MyGreeter greeter;

            @Inject
            Baz(Greeter greeter) {
                this.greeter = greeter;
            }

            public String retrieveGreeterBless() {
                return greeter.sayHi();
            }
        }
as above example MyGreeter injected at `Baz` class's constructor,
and we can call retrieveGreeterBless() to print `MyGreeter` welcome message: hello from MyGreeter


at most ONE constructor can be annotated with @Inject annotation,
if more than one annotated then the bean resolution will fail and DojoContextInitException exception will be thrown.
