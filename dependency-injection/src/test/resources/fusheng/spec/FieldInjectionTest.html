a managed bean can be injected by filed 
    
        @DojoComponent
        public class MyGreeter{
            public String sayHi(){
                return "hello from MyGreeter";
            }
        }

        public class Bar {
            @Inject
            private MyGreeter greeter;

            public String retrieveGreeterBless() {
                return greeter.sayHi();
            }
        }
as above examples MyGreeter injected at `Bar` class's filed,
and we can call retrieveGreeterBless() to print `MyGreeter` welcome message: hello from MyGreeter


a class can declare multiple dependency by Filed injection point 
    
        @DojoComponent
        public class MyGreeter{
            public String sayHi(){
                return "hello from MyGreeter";
            }
        }

        public class Bar {
            @Inject
            private MyGreeter greeter;

            @Inject
            private Foo foo;

            public String retrieveCombinedMessage() {
                return greeter.sayHi() + " AND " +foo.sayHi();
            }
        }
as above examples `MyGreeter` and `Foo` injected at `Bar` class's filed,
and we can call retrieveCombinedMessage() to print welcome message from two dependency: hello from MyGreeter AND hello from Foo
